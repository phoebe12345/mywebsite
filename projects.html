Sweatsink

SweatSink is an Android application crafted in Kotlin, designed to revolutionize your fitness routine by integrating calendar, maps, and chat functionalities into one comprehensive platform. This app allows users to effortlessly plan and track workouts, customize running routes with integrated mapping technology, and log their fitness achievements. By syncing with your personal calendar, SweatSink ensures that your exercise plans fit seamlessly into your schedule, optimizing your time for maximum efficiency. Tailored for the Android ecosystem and built with the robustness of Kotlin, SweatSink stands out as a dynamic tool for anyone looking to elevate their fitness journey.

Yelp Review System

This Java and SQL-based program, developed in Eclipse IDE, offers a robust platform for users to log in, search for businesses and other users, and establish connections within a community-oriented interface. It features advanced search capabilities with customizable filters for businesses and users, displaying results in an organized manner based on specified criteria such as name, city, and rating. Birthday Lookup

This C project serves as an efficient birthday information manager, employing advanced sorting algorithms to organize and present birthday records in a user-friendly, menu-driven interface. It allows for the importing of birthday data from files, displaying entries sorted by date or name, and conducting precise date-based searches, all while efficiently managing data with effective sorting techniques. The program also adeptly handles non-existent files, ensuring a smooth user experience even when errors occur.

Birthday Lookup 2.0

This C++ project is designed as a comprehensive birthday management system, offering users a suite of options for interacting with birthday records. The program enables users to import birthday data from specified files, with robust error handling for non-existent files, and reports the number of entries loaded. It features functionality to display birthdays sorted by month and day, conduct searches for birthdays on specific dates, and add new birthday entries while checking for duplicates. Additionally, users can save the current list of entries to a file, with the option to overwrite existing files. The program concludes by gracefully thanking the user before termination, ensuring a user-friendly experience throughout its operation.

Blackfoot Language App

The "Blackfoot Language App" is an innovative mobile application developed at Simon Fraser University, designed to preserve and revitalize the endangered Blackfoot language. Utilizing a combination of engaging visuals and clear audio recordings, the app offers an interactive and enjoyable learning experience. By integrating these elements, it aims to maintain user engagement and make the process of learning the Blackfoot language both fun and educational. Built on a Python framework, this app represents a crucial step towards cultural preservation and language revitalization, making it an essential tool for both native speakers and those interested in indigenous languages.

Chart Analyzer

This program specializes in analyzing music chart data, offering a range of functionalities including sorting and searching through songs based on various criteria. It can locate songs containing the word "love" in their titles, identify tracks positioned at the top ranks, and pinpoint those advancing in rank each week. Additionally, it calculates the average duration songs remain on the chart and warmly welcomes users by outlining its dual-part structure, ensuring all relevant details are displayed for each search query.

Class Average Calculator

This Python-based program processes a class list to calculate and return the class average, identifying both the lowest and highest scores. It provides a detailed analysis of student performance, offering insights into the overall academic standing of the class.

Transaction Tracker

This program analyzes a mock credit card transaction history to reveal total payments made, identifying the merchant with the highest transaction frequency. It also determines the most spent category, days without spending, and the percentage of transactions under $50, offering a comprehensive overview of spending patterns.
